How It Works

  1. User creates issue with label "research" and topic
  2. GitHub Action triggers automatically on issue creation/labeling
  3. Agent scrapes arXiv, GitHub, HackerNews, Reddit, Dev.to, RSS feeds
  4. LLM synthesizes findings using Groq/Gemini/HuggingFace
  5. Generates outputs in Markdown, JSON, HTML, BibTeX, CSV, Mermaid
  6. Posts summary to issue comment with key findings
  7. Uploads artifacts for download (90-day retention)
  8. Deploys dashboard to GitHub Pages for visualization
